\doxysubsection{Detailed Description}
This class makes up the fundamental rendering unit of \mbox{\hyperlink{classAnsiGL_1_1Surface}{Ansi\+G\+L\+::\+Surface}}. 

A \mbox{\hyperlink{classAnsiGL_1_1Pixel}{Pixel}} is effectively an \mbox{\hyperlink{classAnsiGL_1_1achar}{Ansi\+G\+L\+::achar}} that utilizes a color palette instead of storing the color information directly.
