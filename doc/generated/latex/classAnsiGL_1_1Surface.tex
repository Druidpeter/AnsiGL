\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classAnsiGL_1_1Surface}{Surface}} is a collection of pixels, arranged logically in a rectangular format akin to any conventional rendering context for any other graphics library. The pixel resolution of Ansi\+GL is (as expected, I\textquotesingle{}m sure) a single term character.
