\doxysubsection{Detailed Description}
An A\+N\+S\+I-\/\+Enabled U\+T\+F-\/8 character. 

An \textquotesingle{}achar\textquotesingle{} is simply a standard \textquotesingle{}char\textquotesingle{} paired with a list of A\+N\+SI codes. This is absolutely the most basic element of Ansi\+GL. Even though no other classes are directly based upon achar, all other Ansi\+GL classes will be affected by changes to this class.
